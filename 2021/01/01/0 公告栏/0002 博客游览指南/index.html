博客游览指南（For 2021.01.01）

Date: Jan 1, 2021  00:00:00 Category: 公告

关于 2021-1-1 更新的通知：考虑到简化博客的要求和技术上的难题，在此将博客的主题更换为更为简洁的 chic。如有疑问可以咨询我。

致各位友人：

  你们好，欢迎到访本站！本站以学习与生活为主题。关于本站的建站缘由已在 《Why Blog》 中说过了，所以长话短说：

1.关于网站的架构

  架构如下：

  网站的核心是文章，而所有文章分为五类：公告、札记、论文、随笔、杂文。其中：

（1）公告

关于博客的公告。

（2）札记

  目前正在进行的主题阅读是「西方文化经典主题阅读」：

  • 《荷马史诗》
  • 古希腊戏剧选读
  • 《历史》《伯罗奔尼撒战争史》
  • 《柏拉图对话集》选读
  • 亚里士多德著作

（3）论文

  关于「论文」类文章，我的定位是普通人对人文社科现象的思考，是非专业人士对专业领域的较深入地探究尝试。它不会像学术论文一样深奥，相应的，也不会像学术论文那样精确。我曾经否定过这种文章的价值，因为毕竟已有专业人士珠玉在前了。但我又还是确立了这个项目，一是因为根据我自身经验来看，绝大部分人并不具备天才般的能力——能够不经过学术训练而直接洞察事物的规律，而需要有一段从常人水平到专业水平的衔接阶段；二是因为我认为到访诸君应有较强的批判性思维的能力了，能够去芜存菁地浏览文章。所以我的文章的作用就在这里：为还未能进阶到专业水平的人们提供一个参考的地方；为对相关学科感兴趣但不会往专业化方向发展的人们提供一个交流的地方。至于文章将会出现的谬误，我是这样看待的：人生活在人文环境与社会环境之中，就必定会对于种种现象有所疑惑和思考。虽然但即使是盲人摸象式的虚心求索，也好过坐井观天般的固步自封。对此，欢迎交流与斧正。

（4）随笔

  嗯~，好像没什么价值，主要是与朋友们分享生活的感受和想法。

（5）杂文

  暂未开设。


2.关于网站的关注

  两种方式：

  （1）通过 RSS 订阅网站。

  （2）加入书签。

  另，推荐 PC 端使用以获得更优浏览体验。What’s more, since the server is built on US, ‘technology’ is better.


3.关于网站的更新

  目标：每月一号必然更新一次（一般是在晚上）；每月十五号或然更新一次；其余时间则视情况而定。


4.关于文章的标点、排版和格式

  对于“”和「」的用法：考虑到标点表意的匮乏，兼用“”和「」。其中 “” 用以标示具有讽刺、反语、代称意义的词语， 「」 用以标示引用语。

  文章排版主要采用段落间距分割段落。

  对于论文类文章中的参考资料，将分为两类：一类是参考文献（reference），是学术文章，属于可信来源；一类是参考文章（relation），非学术文章，但仍有不少真知灼见。


5.关于疑问和交流

  如果有疑问或想要交流，可以在「关于」页面中找到我的联系方式。我确信，友善、深刻的交流是分享的最好回报。


  祝你在此获得美的体验和深刻的思考。


渊和 谨献
